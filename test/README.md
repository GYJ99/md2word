# md2word 测试文件说明

本目录包含 md2word 工具的测试文件。

## 测试文件

### 主要测试文档

- **`sample.md`** - 原始完整功能测试文档，包含所有特性
- **`comprehensive_test.md`** - 综合测试文档，整合了所有测试用例

### 测试资源

- **`large.png`** - 大图片测试资源 (1200x800)
- **`small.png`** - 小图片测试资源 (100x100)

## 输出文件

- **`output.docx`** - 基础输出示例
- **`sample_final_fixed.docx`** - 完整功能测试的最终输出
- **`comprehensive_test.docx`** - 综合测试文档的输出

## 使用方法

### 基础测试
```bash
./md2word -i test/sample.md -o test/output.docx -c config.yaml
```

### 综合测试
```bash
./md2word -i test/comprehensive_test.md -o test/comprehensive_test.docx -c config.yaml
```

## 测试覆盖

综合测试文档包含以下功能验证：

1. **自动编号标题** - Word自动编号支持
2. **基础格式** - 粗体、斜体、删除线、行内代码
3. **列表功能** - 有序、无序、嵌套列表
4. **表格转换** - 复杂表格和对齐方式
5. **代码高亮** - 多语言语法高亮
6. **数学公式** - 行内和块级公式，智能尺寸适配
7. **流程图** - Mermaid图表，高清渲染
8. **图片处理** - 自动缩放和优化
9. **链接转换** - 超链接支持
10. **引用格式** - 引用块处理
11. **特殊字符** - Unicode和符号支持

## 验证结果

转换完成后，可以在 Word 或 WPS 中打开生成的 .docx 文件，验证：

1. **自动编号**：标题显示为可编辑的自动编号
2. **格式正确**：字体、字号、行距符合配置
3. **图片显示**：图片正确显示且尺寸合适
4. **表格边框**：表格有正确的边框和对齐
5. **代码高亮**：代码块有语法高亮和背景色
6. **公式渲染**：数学公式显示为清晰的图片，尺寸适中
7. **流程图**：Mermaid 图表高清渲染
8. **超链接**：链接可以点击跳转

所有功能都经过优化，确保在Word文档中的最佳显示效果。
